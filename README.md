##  :watermelon: 项目介绍
此项目为[饥荒管理平台(DMP)](https://github.com/miracleEverywhere/dst-management-platform-api)的前端页面    
该项目build完成后生成的静态资源放入DMP项目中的embedFS/dist目录中使用
